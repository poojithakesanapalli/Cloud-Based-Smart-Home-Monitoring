# 🏠 Cloud-Based Smart Home Monitoring & Automation Platform

A cloud-based IoT Smart Home Monitoring & Automation Platform developed using AWS Cloud services. This project enables users to monitor and control smart home appliances through a modern web dashboard while leveraging cloud technologies for storage, monitoring, and automation.

---

## 📌 Project Overview

The Cloud-Based Smart Home Monitoring & Automation Platform is designed to simulate a real-world smart home environment. Users can control appliances such as lights, fans, doors, and monitor temperature through an interactive dashboard.

The project demonstrates how cloud computing can be integrated with IoT concepts to build scalable, secure, and efficient home automation systems.

---

## 🎯 Objectives

- Develop a cloud-based smart home dashboard.
- Monitor home appliances remotely.
- Control smart devices using a web interface.
- Store device information securely in the cloud.
- Demonstrate AWS cloud services integration.
- Provide a scalable and secure architecture.

---

## 🚀 Features

- 💡 Control Living Room Lights
- 🌀 Control Ceiling Fan
- 🌡️ Monitor Temperature
- 🚪 Smart Door Lock Status
- 🔒 Motion Detection Alert
- ☁️ Cloud Storage using Amazon S3
- 🗄️ Data Storage using Amazon DynamoDB
- 💻 Hosting using Amazon EC2
- ⚡ Serverless Automation using AWS Lambda
- 📊 Cloud Monitoring using AWS CloudWatch
- 🔐 Secure IAM Role Management
